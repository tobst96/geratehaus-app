# Planner

Vor jeder größeren Änderung:

## Analyse

- Code lesen
- ähnliche Funktionen suchen
- betroffene Dateien finden
- Risiken erkennen

Keine Änderungen durchführen.

Backend

Frontend

Migration

API

Tests

Review

- Reihenfolge
- Aufwand
