# Feature Implementation

Vor dem Implementieren

- bestehenden Code analysieren
- vorhandene Patterns übernehmen
- keine neuen Patterns einführen

Während der Umsetzung

- möglichst kleine Änderungen
- keine Refactorings
- nur Dateien ändern, die wirklich betroffen sind

Nach der Umsetzung

- pytest
- npm run build
- neue Tests
- Änderungen zusammenfassen
- Todo auf erledigt setzen `todo.md` des Skills `todo`
